# Nodeblog-Node博客系统
全栈开发，使用yoeman自动化构建工具搭建一个完整的项目的结构<br/>
主要有三部分组成：Yo（脚手架，自动化工具），grunt/**gulp**（构建工具，压缩js/css/图片等，等你挖掘），bower（包管理工具，大家熟悉的dependencise）
